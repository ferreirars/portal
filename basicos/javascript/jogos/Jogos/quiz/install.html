How to install the CWE Quiz Program

Please read the following page carefully and in detail to ensure an error free installation of the quiz program. It can be used on a website which is online (on the WWW), or one which is offline (such as a company intranet).

I.   Prerequisites
A text editor.
HTML knowledge is helpful.
II.  Required Downloads
The CWE Quiz program, version 1
As a result, you should have 5 files: index.html, install.html, edit.html, writeinput.js & write.js
III.  The online builder
To make it easy to create a quiz, we designed a builder for you. This can be found here. (The quiz builder must be used online to work - it uses server side scripting)

You can add a question, followed by its choices and select the right answer and its explanation. There is no limit to how many questions or answer choices you can add, but there is a minimum of two questions and two answer choices per question.

We think the builder should be fairly self-explanatory - so try it out!.
IV.  The editor
The edit.html file is to edit the quiz once it has been created, not to create new ones. You can edit your data offline, but then connect to our website to re-generate the file in the correct format and re-download the data file. It still requires server-side scripting for this process. Editing the data file itself is not recommended unless you know advanced JavaScript!
V.  Linking to the quiz
So that you can have multiple quizzes, you can download more than one quiz file from the online builder, without needing to re-download more copies of the quiz program. To enable this, you must link to each quiz differently in the following way:

<a href="index.html?file=quiz.js">Quiz title</a>

Just change the section in red to the name of the quiz data file that you downloaded using the online builder.
VI.   Using on a website
On the website, you need to put all the files that were downloaded into the same folder. Then link to the quizzes as outlined above.
VII.  A few cautionary tales
Do not edit any of the JavaScript and HTML code that the quiz uses to generate the question and answer display. If you have knowledge of HTML/CSS, you can change the formatting, but please use caution. The code in this program is complicated, and can easily be corrupted if you are careless when making changes.
VIII. Troubleshooting

If did not find a solution to the problem here, please contact us.
